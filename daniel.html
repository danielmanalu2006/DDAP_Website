FILKOM Student Portal

Created on May 15, 2023 Web App
A
Ahmad Fauzan

Project Description

FILKOM Student Portal is a comprehensive web application designed to streamline academic processes for students at the Faculty of Computer Science, Universitas Brawijaya. The portal serves as a centralized platform for students to access course materials, submit assignments, track their academic progress, and communicate with instructors.

The application features a modern, intuitive interface that prioritizes user experience, making it easy for students to navigate through various sections. The dashboard provides a quick overview of upcoming deadlines, recent announcements, and academic performance metrics.

Key features include:

  • Personalized dashboard with academic calendar and notifications
  • Course management system with material downloads and assignment submissions
  • Real-time grade tracking and academic progress visualization
  • Integrated messaging system for student-instructor communication
  • Mobile-responsive design for access on various devices

The project was developed using React.js for the frontend, Node.js and Express for the backend, and MongoDB for the database. Authentication is handled securely using JWT tokens, and the application implements role-based access control to manage different user privileges.

Technologies Used

React.js Node.js JavaScript MongoDB Express Tailwind CSS JWT Chart.js
